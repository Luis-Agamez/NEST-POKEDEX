# Ejecutar en modo de desarrollo

1. Clonar el repositorio 
2. Ejecutar

```
yarn insttall
```

3. Tener Nest CLI instalado 

```
npm i -g @nestjs/cli
```

4.Levantar la base de datos

```
docker-compose up -d
```
